# PAGINA WEB ANIME XTREMO

## DESCRIPCIÓN DEL PROYECTO

PAGINA WEB DE TIENDA DE PRODUCTOS DE ANIME DESARROLLADA COMO PROYECTO ACADÉMICO.
EL SISTEMA PERMITE VISUALIZAR PRODUCTOS, ADMINISTRARLOS DESDE UN PANEL DE ADMINISTRACIÓN Y ALMACENAR LA INFORMACIÓN EN UNA BASE DE DATOS.

EL PROYECTO UTILIZA UNA ARQUITECTURA SEPARADA ENTRE FRONTEND Y BACKEND PARA MEJOR ORGANIZACIÓN Y ESCALABILIDAD.

---

## TECNOLOGÍAS UTILIZADAS

### FRONTEND

ANGULAR
HTML
CSS
TYPESCRIPT

### BACKEND

NODE.JS
EXPRESS

### BASE DE DATOS

MONGODB

---

## FUNCIONALIDADES DEL SISTEMA

✔ VISUALIZACIÓN DE PRODUCTOS DE ANIME
✔ PANEL DE ADMINISTRACIÓN
✔ REGISTRO DE PRODUCTOS
✔ EDICIÓN DE PRODUCTOS
✔ ELIMINACIÓN DE PRODUCTOS
✔ SUBIDA DE IMÁGENES DE PRODUCTOS
✔ API REST PARA GESTIÓN DE PRODUCTOS
✔ CONEXIÓN A BASE DE DATOS MONGODB

---

## ESTRUCTURA DEL PROYECTO

BACKEND
CONTIENTE EL SERVIDOR, LAS RUTAS, LOS MODELOS Y LA CONEXIÓN A LA BASE DE DATOS.

FRONTEND
CONTIENTE LA INTERFAZ DESARROLLADA EN ANGULAR PARA LA INTERACCIÓN DEL USUARIO.
